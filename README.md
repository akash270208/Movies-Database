# Movies-Database

## Description 
I have this project to save movies details. Following details can be saved.

1. Insert new record
2. Display record.
3. Update existing record
4. Delete any record.

## Language used -
Python

## Library Used - 
SQLite (to store the values to database).
Tkinter (to create user interface).
